# 入力サウンドプラグイン
入力検知で効果音を鳴らすプラグインです  

## ダウンロード方法
[ここからダウンロード](https://github.com/HAIsanGames813/InputSoundPlugin/releases)
### 何度も言いますがネタプラグインです。入れても後悔しないでください。
設定からOFFにはできるので安心してください

## 使用方法
設定から入力サウンドタブを開き音声を設定してください
